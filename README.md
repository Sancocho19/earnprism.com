# EarnPrism site V7

This package includes the V7 build of EarnPrism with the full calculator hub, the expanded guide hub, trust pages, and a larger long-tail cluster built around high-intent hourly wage and salary benchmark searches.

## Main sections
- Home
- 16 calculators
- 32 guides
- Methodology
- Editorial standards
- Contact
- Privacy
- Terms

## V7 additions
- extra long-tail guide pages for common wage and salary searches
- stronger guide hub with more search coverage
- updated sitemap
- improved launch notes for Search Console and indexing work

## Before going live
- confirm that `support@earnprism.com` is the email you want displayed
- add any real analytics, ads, or affiliate providers to the privacy page once they are active
- connect your real domain in Vercel
- submit the sitemap in Google Search Console

## Notes
- The site uses clean URLs through `vercel.json`.
- Calculator history is stored locally in the browser.
