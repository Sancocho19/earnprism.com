# IncomeGrid — Tier 1 static starter

This is a static website starter designed for Vercel.

## What it is
A compact calculator hub focused on salary, freelance pricing, creator income, raises, budgets, and savings.

## Why this version is different
This version is written in English on purpose so it is better aligned with Tier 1 traffic and higher-value monetization paths.

## Deploy
1. Replace `https://www.yourdomain.com` with your real domain across the project.
2. Push the folder to GitHub.
3. Import the repo into Vercel or deploy with the CLI.
4. Add your custom domain.
5. Submit `/sitemap.xml` in Google Search Console.

## Important
- Update privacy and terms with real business details.
- Add analytics and consent flow if required in your target region.
- Expand carefully after the first pages are indexed.
