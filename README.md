# EarnPrism Site v3

Static site for Vercel with clean URLs.

## Included
- 12 calculators
- 9 guides
- Home, About, Privacy, Terms, 404
- `robots.txt`, `sitemap.xml`, `vercel.json`

## Deploy
1. Upload the contents of this folder to a GitHub repo.
2. Import the repo into Vercel.
3. Leave the preset as **Other**.
4. Root directory should be `./` if these files are in the repo root.
5. No environment variables are required.

## Final launch checklist
- Replace placeholder legal language with final business details.
- Confirm `earnprism.com` is the final domain in Vercel and DNS.
- Submit the sitemap in Google Search Console.
- Add analytics only after you know which vendor you want to use.
